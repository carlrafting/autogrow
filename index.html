autogrow makes textarea elements more elegent & less frustrating, by making them autogrow.

Quick Start

Here's an example of how to create an expanding textarea like the demo above.

<textarea id="autogrow"></textarea>
autogrow('autogrow'); // get textarea element with id="autogrow"

Documentation

You can check out documentation on how to use autogrow, and what autogrow is on GitHub.

Install

Add autogrow to your project by either cloning the repository, or download the latest release as a ZIP file from GitHub.

git clone https://github.com/carlrafting/autogrow.git

It's also possible to download the files directly below.
